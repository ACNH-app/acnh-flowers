# ACNH 꽃 교배 정리


## 1. 교배꽃 기본 규칙

- 물 주기 필수: 비나 눈이 오지 않는 날에는 반드시 물을 줘야 다음 날 확률적으로 새로운 꽃이 피어납니다.
- 상점 씨앗으로 시작하기: 섬 자생 꽃이나 마일섬 꽃은 숨은 유전자가 섞여 있을 수 있으니, 정확한 교배를 원하면 늘봉이나 너굴상점의 `씨앗`으로 시작하는 것이 좋습니다.
- 공간 확보: 꽃 주변 8방향 중 최소 한 칸은 비어 있어야 새로운 꽃이 자랄 수 있습니다.

### 꿀팁

- 통신(멀티) 활용하기: 다른 섬 유저가 놀러 와서 물을 주면 교배 확률이 크게 올라가며, 최대 5명까지 효과가 중첩됩니다.
- 친구가 물을 준 꽃은 더 크고 반짝이는 이펙트가 생겨 구분하기 쉽습니다.
- 아름다운 섬 조례: 안내소의 여울이에게 `아름다운 섬 조례`를 부탁하면 내가 물을 주지 않아도 꽃이 더 자주 번식합니다.
- 다만 아름다운 섬 조례는 원하지 않는 꽃까지 많이 늘릴 수 있으니 관리에 주의가 필요합니다.

## 2. 꽃 교배의 기본 원리

### 유전자 시스템

모동숲의 꽃은 각각 3~4개의 유전자를 가지고 있으며, 이 조합이 꽃의 색상을 결정합니다. 유전자는 `0`, `1`, `2` 값을 가지며 부모 꽃의 유전자가 자식 꽃에게 전달됩니다.

`0`, `1`, `2`는 각 유전자 자리의 우성/열성 조합을 숫자로 축약한 표기입니다.

| 숫자 | 의미 | 문자 예시 | 설명 |
| --- | --- | --- | --- |
| `0` | 열성 순종 | `aa` | 열성 유전자 2개를 모두 가진 상태 |
| `1` | 잡종 | `Aa` | 우성과 열성을 1개씩 가진 상태 |
| `2` | 우성 순종 | `AA` | 우성 유전자 2개를 모두 가진 상태 |

예를 들어 유전자형이 `201`이라면, 이를 자리별로 풀어서 `AA / aa / Aa`처럼 읽을 수 있습니다.

- 첫 번째 자리 `2` -> `AA` (우성 순종)
- 두 번째 자리 `0` -> `aa` (열성 순종)
- 세 번째 자리 `1` -> `Aa` (잡종)

즉 `201`은 세 개의 유전자 자리에 대해 `AA-aa-Aa` 조합을 뜻한다고 보면 됩니다. 장미처럼 유전자가 4자리인 꽃은 같은 방식으로 `2001`, `1110`처럼 한 자리씩 끊어서 읽으면 됩니다.

> 복잡한 유전자를 직접 계산할 필요는 없습니다. 아래 레시피처럼 씨앗부터 차근차근 교배를 시작하면 원하는 색상을 훨씬 안정적으로 얻을 수 있습니다.

### 교배 배치 방법

두 꽃이 교배하려면 대각선을 포함한 인접 8칸 안에 위치해야 합니다. 물을 준 뒤 다음 날 아침, 주변 빈 칸에 새 꽃이 생성되면 교배 성공입니다.

1. 교배시킬 두 꽃을 대각선 또는 나란히 배치합니다.
2. 두 꽃 주변에 자식 꽃이 생성될 빈 칸을 확보합니다.
3. 매일 물을 주고 다음 날 아침 새 꽃이 생겼는지 확인합니다.
4. 원하는 결과물이 나오면 그 꽃을 활용해 다음 단계 교배를 진행합니다.

### 주의 사항

- 자가복제 vs 교배: 교배 상대가 없으면 꽃은 자신과 같은 꽃을 복제할 수 있습니다. 교배가 목표라면 짝이 되는 두 꽃만 깔끔하게 배치하는 것이 중요합니다.
- 잡초·클론 꽃 관리: 목표하지 않은 꽃이나 복제 꽃이 빈 칸을 막으면 원하는 꽃이 나올 확률이 줄어듭니다.

## 3. 가장 효율적인 배치 방법

꽃을 심을 때는 서로 인접하면서도 새 꽃이 자랄 공간이 남는 대각선(격자/체스판) 배치가 가장 효율적입니다.

### 추천 배치

```text
🌸 ⬜ 🌸
⬜ 🌸 ⬜
🌸 ⬜ 🌸
```

`🌸 = 꽃`, `⬜ = 빈 땅`

이렇게 배치하면 꽃끼리 서로 닿으면서도 주변에 새 꽃이 자랄 공간이 충분히 확보됩니다.

## 4. 친구 방문 보너스

다른 플레이어가 내 섬을 방문해서 꽃에 물을 주면, 다음 날 교배 성공 확률에 추가 보너스가 붙습니다.

| 물을 준 친구 수 | 다음 날 교배 성공률 (대략) |
| --- | --- |
| 0명 (본인만) | 약 5% |
| 1명 | 약 25% |
| 2명 | 약 50% |
| 3명 | 약 75% |
| 4명 | 약 80% |
| 5명 이상 | 약 80% (최대) |
